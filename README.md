# バリュー株スクリーナー

東証プライム全銘柄をPER/PBRでスクリーニングできるWebアプリ。
J-Quants API連携でリアルタイムデータを取得。

## セットアップ

### 1. 依存パッケージのインストール

```bash
cd value-screener
npm install
```

### 2. 環境変数の設定

`.env.local` ファイルにAPIキーが設定済みです。
（本番運用時は `.env.local` をGitにコミットしないでください）

### 3. 開発サーバーの起動

```bash
npm run dev
```

ブラウザで http://localhost:3000 を開く

## 機能

- **スクリーニング**: PER/PBR上限スライダーで全銘柄をフィルタリング
- **セクター絞り込み**: 33業種から選択
- **ウォッチリスト**: 銘柄を保存し、登録日からのパフォーマンス追跡
- **AI分析連携**: 🤖ボタンでグレアム流の詳細分析プロンプトをコピー → Claudeに貼り付け

## J-Quants プラン別の違い

| プラン | 月額 | データ遅延 |
|--------|------|-----------|
| Free | 無料 | 12週間遅れ |
| ライト | ¥1,650 | リアルタイム |
| スタンダード | ¥3,300 | リアルタイム＋先物 |

リアルタイムデータが必要な場合はライトプラン以上に。

## デプロイ（Vercel）

1. GitHubにリポジトリを作成してpush
2. Vercelでリポジトリを連携
3. 環境変数 `JQUANTS_API_KEY` を設定
4. デプロイ完了

## 技術スタック

- Next.js 14 (App Router)
- J-Quants API v1
- React 18
